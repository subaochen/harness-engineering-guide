### Step 2：复制 HTML 内容

2. 按 **Ctrl + C** 复制

**检查**：

---

### Step 3：粘贴到公众号编辑器

**在公众号后台**：
1. 访问 https://mp.weixin.qq.com/
2. 点击左侧 **"内容与互动"** → **"草稿箱"**
3. 找到目标文章（已有标题和作者）
4. 点击 **"编辑"**
5. 在编辑器空白区域点击一下
6. 按 **Ctrl + V** 粘贴
7. **等待 5-10 秒**（内容会逐步加载）

**检查**：
- [ ] 标题格式正常（蓝色大标题）
- [ ] 代码块有背景色（深灰色）
- [ ] 表格显示正常（蓝白相间）
- [ ] 引用块有蓝色左边框
- [ ] 卡片样式正常（渐变背景）
- [ ] 分隔线显示正常（蓝紫渐变）

---

### Step 4：上传封面图

**在公众号编辑器**：
1. 点击 **"封面图"** 区域
2. 选择 **"本地上传"**
3. 选择文件：`covers/<文章名>-cover.jpg`
4. 裁剪为 **2.35:1**（公众号封面标准）
5. 点击 **"确定"**

**检查**：
- [ ] 封面图清晰（1920x1080）
- [ ] 裁剪后文字完整
- [ ] 图片质量良好

## 🎯 批量发布技巧

### 一次准备多篇文章
```bash
# 生成多篇 HTML
for article in article1 article2 article3; do
  bun ~/.openclaw/skills/baoyu-post-to-wechat/scripts/md-to-wechat.ts \
    --markdown docs/${article}.md \
    --output docs/wechat-articles/${article}.html \
    --theme default \
    --color blue
done

# 生成多张封面图
for article in article1 article2 article3; do
  bun ~/.openclaw/skills/baoyu-image-gen/scripts/main.ts \
    --promptfile covers/${article}/prompts/cover.md \
    --image covers/${article}-cover.jpg \
    --ar 16:9 \
    --quality 2k
done
```

## ⚠️ 常见问题

### Q1: 粘贴后内容不显示？
**原因**：公众号编辑器加载慢  
**解决**：等待 10-15 秒，不要重复粘贴

### Q2: 代码块格式错乱？
**原因**：HTML 兼容性问题  
**解决**：使用 baoyu-post-to-wechat 技能重新转换

### Q3: 封面图上传失败？
**原因**：图片太大或格式不对  
**解决**：
- 确保图片 < 5MB
- 使用 JPG 格式
- 尺寸 1920x1080 或更小

### Q4: 手机预览和电脑显示不一样？
**原因**：公众号手机端和 PC 端渲染差异  
**解决**：以手机预览为准，必要时调整 HTML
